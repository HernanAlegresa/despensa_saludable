# Imágenes de la página "Quiénes somos"

Subí aquí las imágenes que usa la página **Quiénes somos**. La carpeta es:

**`public/quienes-somos/`** (dentro del proyecto)

## Archivos a subir

1. **Hero** (primera sección: "Despensa Natural" con el texto sobre la imagen)
   - Nombre del archivo: **`hero.jpeg`** (o `hero.jpg`)
   - Imagen de fondo full-bleed, como en la homepage.
   - Si usás `.jpg`, avisá para actualizar la ruta en el código.

2. **Nuestra historia** (sección con el texto "Nuestra historia")
   - Nombre del archivo: **`nuestra-historia.jpeg`**
   - Se muestra a la izquierda de ese bloque.

3. **Qué nos guía** (sección "Mascotas")
   - Nombre del archivo: **`que-nos-guia.jpeg`**
   - Se muestra a la derecha de esa sección.

## Cómo subir

- Copiá o arrastrá los archivos a la carpeta **`public/quienes-somos/`** del proyecto.
- Asegurate de que los nombres sean exactamente los indicados arriba.

Cuando los archivos estén en esa carpeta, la página ya los usará automáticamente.
