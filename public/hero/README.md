# Imagen del Hero (homepage)

Subí aquí la imagen de fondo del hero de la homepage.

**Archivo principal (escritorio):**  
- Nombre del archivo: **`hero.jpg`**  
- Ruta completa: `public/hero/hero.jpg`  
- Formatos aceptados: .jpg, .jpeg, .webp o .png  

**Opcional — versión móvil:**  
- Nombre del archivo: **`hero-mobile.jpg`**  
- Ruta completa: `public/hero/hero-mobile.jpg`  
- Si no subís esta imagen, se usará la misma que en escritorio.

Cuando estén subidas, avisá para conectar las imágenes en el componente Hero.
