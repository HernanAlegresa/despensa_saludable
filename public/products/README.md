# Imágenes de productos

Cada subcarpeta corresponde a un producto del catálogo (el nombre de la carpeta es el **slug** del producto).

**Qué hacer:** En cada carpeta, subí la imagen del producto con el nombre **`main.jpg`** (o `main.webp` / `main.png` si preferís otro formato).

Ejemplo:
- `aceite-oliva-campo-claro-500ml/main.jpg`
- `ghee-agni-170g/main.jpg`
- etc.

Cuando todas las imágenes estén cargadas, avisá para actualizar el catálogo en el código.
